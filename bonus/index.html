Condividi facilmente qualunque cartella con chi vuoi

La condivisione di file con Dropbox è rapida e semplice. Ti basta un link per inviare qualunque cosa, da foto e video a cartelle compresse e grandi file CAD, a chiunque, anche se non ha un account Dropbox. Condividi link e file da qualsiasi posizione usando telefono, tablet o computer. E, a differenza degli allegati delle email, non devi preoccuparti dei limiti di dimensione dei file: condividi anche file che corrispondono alla tua capacità massima di archiviazione (da 2 GB in su).

Ricevi aggiornamenti in tempo reale sui file condivisi

Scopri esattamente quando qualcuno ha apportato delle modifiche a un file. Con la condivisione di file sul cloud, vedrai immediatamente chi ha aggiunto, modificato, eliminato, rinominato o spostato un file. Qualunque modifica apportata a un file condiviso sarà automaticamente aggiornata e sincronizzata per tutti i destinatari.
